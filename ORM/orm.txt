ORM
Object Relational Mapper

-> It's a library (that you install)
    -> prevents SQL injection
    -> generates SQL code for you

There are 10+ of ORMs available for all kinds of databases.

-> SQL-type of DBs
    => Sequelize (popular)
    => Knex.js
    => TypeORM
    => Prisma (popular)
    => ...



==> ORM manage the SQL command / syntax -> It has their own function so you just use it, instead of creating you own SQL command <==
